About:<br>
SampleImagesLayout loads images in grid format from JSON file on page load.<br>
SampleImagesLayout.png provided as example of view within browser.<br>
<br>
To deploy and view in apache tomcat:
1) Copy SampleImagesLayout folder to tomcat\webapps folder.
2) Start tomcat.
3) Launch browser.
4) Browse to 'http://localhost:8080/SampleImagesLayout'
<br>
Assumptions:<br>
Apache Tomcat is configured to run on default port on local host.<br>
